기술 개발 배경
수어를 잘 모르는 사람이 많기에 농인들이 수어를 사용했을 때 농인과 일반인 사이 소통을 수월하게 도울 수 있도록 수어-텍스트 변환 통역 서비스를 구상함.

기대효과
[기술적 기대효과]
- 사람의 몸짓에 따른 관절 위치 벡터 분석
- 관절 위치를 학습시킨 수어와 대응하여 결과 도출
- 농인들의 의사소통에 편리함을 부여

[사회적 기대효과]
- 음성 언어를 사용하는 청인이 다수인 사회 속에서 시각 언어인 수어를 통해 삶을 영위하는 농인들이 본 프로그램을 통해 원활한 소통 가능.
